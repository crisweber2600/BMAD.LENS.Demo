# Lens Compass Workflows

## Core Workflows

- **onboarding** — Capture identity, roles, and initial preferences
- **next-step-router** — Suggest the most relevant next workflow
- **roster-tracker** — Auto-log contribution depth to the roster

## Feature Workflows

- **profile-manager** — View or edit personal profile
- **roster-query** — Find who worked on a domain/service/microservice
- **preference-learner** — Persist learned preferences
- **context-analyzer** — Determine current phase and context

## Utility Workflows

- **data-cleanup** — Reset personal or roster data
- **roster-export** — Generate roster reports
