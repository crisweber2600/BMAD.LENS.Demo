# Step: Generate Propagation Plan

**Goal:** Create report-only propagation plan with conflict details and actions.

---

## Plan Template

```markdown
# Amendment Propagation Plan

**Parent Constitution:** {layer}/{name}
**Amendment:** #{amendment_id}
**Date:** {today_date}
**Version Change:** {old_version} → {new_version}
**Bump Type:** {bump_type}
**Bump Rationale:** {bump_rationale}

---

## Amendment Summary

| Field | Value |
|-------|-------|
| Type | {Add/Modify/Clarify/Deprecate} |
| Article | {article_id}: {article_title} |
| NON-NEGOTIABLE | {Yes/No} |
| Change | {description} |

---

## Affected Child Constitutions

{if affected_children_count > 0:}

| Constitution | Layer | Articles | Conflicts | Status |
|-------------|-------|----------|-----------|--------|
{for child:}
| {child_name} | {child_layer} | {article_count} | {conflict_count} | {ALIGNED/⚠️ REVIEW} |
{endfor}

### Detailed Conflict Analysis

{for each child with conflicts:}
#### {child_layer}/{child_name}

**Conflicts Detected ({conflict_count}):**

{for conflict:}
{n}. **{conflict.type}** — {conflict.description}
   - **Parent Article:** {conflict.parent_article}
   - **Child Article:** {conflict.child_article}
   - **Recommended Action:** {get_recommended_action(conflict.type)}
{endfor}
{endfor}

---

## Follow-up Actions

{if total_conflict_count > 0:}
- [ ] Review {conflict_count} child constitution(s) for compatibility
- [ ] Update conflicting articles
- [ ] Re-run compliance checks in affected projects
- [ ] Coordinate with teams owning affected constitutions
{else:}
- [ ] No immediate action required — all children aligned
{endif}

{else:}
No child constitutions affected by this amendment.
{endif}

---

*This is a REPORT-ONLY plan. No constitutions have been auto-modified.*
*Generated by Scribe on {today_date}*
```

**Save at:** `_bmad-output/lens-work/constitutions/{layer}/{name}/propagation-plans/{today_date}-amendment-{amendment_id}.md`

**Commit via Casey** with governance-prefixed message
