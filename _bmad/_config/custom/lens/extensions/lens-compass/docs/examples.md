# Lens Compass Examples

## Example: First-Time Onboarding

- User launches Lens
- Compass detects git user and confirms identity
- User selects role(s)
- Profile is created in `_bmad-output/personal/`

## Example: Role-Based Guidance

- PO completes new-service
- Compass suggests next-step-router
- Compass recommends new-microservice or lens-switch

## Example: Roster Query

- Developer asks "who worked on payments service?"
- Compass reads `.bmad-roster.yaml`
- Returns contributors and depth
